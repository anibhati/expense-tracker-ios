# iOS Expense Tracker

An iOS expense tracking application built with **Swift** that allows users to record, categorize, and monitor their daily spending. The app focuses on simplicity, clean UI, and core personal finance functionality.

## Features
- Add, edit, and delete expenses  
- Categorize expenses (e.g. Food, Transportation, Entertainment, etc.)
- View a list of all recorded expenses
- Track total spending over time
- Clean and intuitive iOS user interface

## Tech Stack
- **Language:** Swift  
- **Platform:** iOS  
- **Framework:** SwiftUI
